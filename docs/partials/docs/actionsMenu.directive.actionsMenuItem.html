actionsMenuItem
directive in module actionsMenu

Description

This directive represent an item into an actions menu.

This directive is included by its parent directive: actionsMenu, and should not be called offside of the module scope.

Usage

as attribute
<ANY actions-menu-item
     actions-menu-item="{ActionMenuItem}"
     actions-menu-item-on-click="{Function}">
   ...
</ANY>

Parameters

ParamTypeDetails
actions-menu-itemActionMenuItem

An instance of ActionMenuItem.

actions-menu-item-on-clickFunction

A callback function called when the action menu item has been clicked.
