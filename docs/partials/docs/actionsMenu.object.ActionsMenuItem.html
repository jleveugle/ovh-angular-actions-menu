ActionsMenuItem
object in module actionsMenu

Description

Factory that describe an item into an actions menu.

Usage

ActionsMenuItem(options);

Parameters

ParamTypeDetails
optionsObject

Options for creating a new ActionsMenuItem instance.

PropertyTypeDetails
externalString

Is the link is an external link. In other words, will we be redirected offside of the manager ?

(default: false)

stateParamsObject

The params to pass to the state.

(default: {})

targetString

The target of the href anchor tag. This will be the target html attribute.

(default: _self)

Methods

  • addSubAction(subActionOptions)

    Add a sub action item to current actions menu item instance.

    Parameters
    ParamTypeDetails
    subActionOptionsObject

    Options for creating an actions menu sub item. See constructor options for more details.

    Returns
    ActionsMenuItem

    The new instance of actions menu item created.

  • getFullSref()

    Get the full sref string that will be applied to item that have state defined.

    Returns
    String

    The string representation of a state. For example: my.manager.state({param1:1, param2:2}).

  • hasSubActions()

    Check if the instance of actions menu item has sub actions defined.

    Returns
    Boolean

    true if item has subItems, false otherwise.

Example

angular.module("myManagerApp").controller("MyTestCtrl", function (ActionsMenuItem) {
    var actionMenuItem = new ActionsMenuItem({
        title: "My Beautiful title",
        icon: "filled-check",
        href: "http://www.google.be"
    });
});
