ActionsMenu
object in module actionsMenu

Description

Factory that describe an actions menu.

A new instance of ActionsMenu is created and used by the actionsMenu directive.

Dependencies

  • ActionsMenuItem

Usage

ActionsMenu(options);

Parameters

ParamTypeDetails
optionsObject

Options for creating a new ActionsMenu instance.

PropertyTypeDetails
actionsMenuItemsArray<Object>

The options of the items that will be added to the ActionsMenu instance.

(default: Empty Array)

Methods

  • addActionItem(actionMenuItemOptions)

    Add an actions menu item into actions list.

    Parameters
    ParamTypeDetails
    actionMenuItemOptionsObject

    The options for creating a new action menu item. See ActionsMenuItem factory for available options.

    Returns
    ActionsMenuItem

    The added actions menu item.

Example

angular.module("myManagerApp").controller("MyTestCtrl", function (ActionsMenu) {
    var actionMenu = new ActionsMenu({
        actionsMenuItems: [{
            title: "My Beautiful title",
            icon: "filled-check",
            href: "http://www.google.be"
        }]
    });
});
